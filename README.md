# 🎬 Movie App

Aplicación moderna de películas desarrollada con **React + TypeScript + TailwindCSS** utilizando la API de TMDB.

## ✨ Features

- 🔥 Películas populares
- 🔎 Búsqueda en tiempo real
- ❤️ Sistema de favoritos
- 💾 Persistencia con LocalStorage
- ⚡ Navegación con React Router
- 🎨 Diseño responsive moderno
- 🍿 UI inspirada en Netflix / Disney+

---

## 🚀 Tecnologías utilizadas

- React
- TypeScript
- TailwindCSS
- React Router DOM
- React Icons
- TMDB API
- Vite
